# Gestor de Tareas en Python

Pequeño programa de consola que permite administrar una lista de tareas.  
Es ideal para practicar bucles, listas, validaciones y funciones en Python.

## Funcionalidades

- Agregar tareas
- Listar todas las tareas
- Eliminar una tarea por número
- Menú interactivo en consola

## Cómo ejecutar

Asegurate de tener Python instalado. Luego, ejecutá:

```bash
python GestorDeTareas.py
